Multi-slot pilot did not show a clear formal vs informal gap overall.

Most formal and informal responses were broadly similar in structure, task framing, and level of detail.
    When response quality issues appeared, they usually appeared in both versions, not only in the informal one.

The only slightly more noticeable difference was in the gift recommendation example: the informal response drifted a bit further from the user constraint.
    Even in that case, the formal response also violated the constraint to a meaningful extent, so the difference was limited.

This slice was still useful as a check, but it did not produce a particularly diagnostic or interesting contrast.

If I revisit multi-slot later, it should use stronger exclusion constraints or more conflict-prone slot combinations.
